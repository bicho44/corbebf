# CORBEB SRL

## Captura de Remitos

Sistema de Simplificación de captura de remitos para reemplazar las planillas.

### Acciones del Sistema (wip)

-   Carga de Usuarios
-   Carga de Clientes
-   Carga de Direcciones
-   Asignaciones de días de reparto
